## About

A fast, responsive personal website built to be easy to extend. Animations use Framer Motion, styling with Tailwind, and contact form via EmailJS. Built with TypeScript and Vite.

## Tech stack

- React (client components)
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- EmailJS
- Deployment: `gh-pages`

## Quick start

Prerequisites:
- Node.js (LTS)
- npm

Install and run locally:

```bash
npm install
npm run dev
npm run build
npm run preview
```

To deploy to GitHub Pages:

```bash
npm run predeploy
npm run deploy
```

## Scripts

- `npm run dev` — start development server
- `npm run build` — create production build
- `npm run preview` — preview production build
- `npm run predeploy` — build before deploy
- `npm run deploy` — deploy `dist` to GitHub Pages

## Features

- Responsive layout and accessible components
- Client-side animations with Framer Motion
- Light / dark theme with a `themeChanged` custom event
- Contact form using EmailJS
- Loader overlay and staged content mounting for smooth entry
- Simple GitHub Pages deployment

## Configuration notes

- EmailJS: current code references `service_2xy699q`, `template_ndty7cf` and a public key. For security and flexibility, move these to environment variables and load them at runtime or via your build process.
- Theme: components listen for a `themeChanged` custom event (detail: `"light"` | `"dark"`). If you change theme logic, keep this event to update components.
- Deployment: homepage is set in `package.json` to `https://JBrunnerhtl.github.io/personal-website2.0/`. Adjust if the repo or user changes.

## Development notes

- Keep code linted and formatted (ESLint + Prettier recommended).
- Components are client-side; many use `useEffect` and should only mount when visible (`DelayedMount` pattern in `src/App.tsx`).
- Verify contact form clears fields after submit. Current form calls `form.current.reset()` — if any visible message nodes persist, ensure you clear their `textContent` or state.
- Remove any autogenerated assistant comments before publishing.
